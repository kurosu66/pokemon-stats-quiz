# pokemon-stats-quiz

ポケモンの種族値を見てポケモン名を当てるCLIクイズゲームです。

## 遊び方

```bash
npx pokemon-stats-quiz
```

1. 出題する世代を選ぶ
2. 表示された種族値（H/A/B/C/D/S）から、どのポケモンが正解か4択から選ぶ
3. 正解・不正解が表示される

## 対応世代

- 第1世代（赤・緑）〜 第9世代（スカーレット・バイオレット）

## 必要環境

- Node.js 18以上

## 使用API

- [PokéAPI](https://pokeapi.co/)

# pokemon-stats-quiz (English)

A CLI quiz game where you guess the Pokémon from its base stats.

## How to play

```bash
npx pokemon-stats-quiz
```

1. Select a generation
2. Guess which Pokémon matches the displayed base stats (HP/Atk/Def/SpAtk/SpDef/Spd) from 4 choices
3. See if you got it right

## Supported generations

- Generation 1 (Red/Green) through Generation 9 (Scarlet/Violet)

## Requirements

- Node.js 18 or higher

## API

- [PokéAPI](https://pokeapi.co/)
